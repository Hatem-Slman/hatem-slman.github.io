واجهة GitHub Pages جاهزة للنشر

حاتم الغامدي

صفحة شخصية احترافية يمكن استخدامها كبداية على GitHub Pages لعرض نبذة مختصرة، الأعمال، وسائل التواصل، أو أي مشروع مستقبلي. تم تصميمها بشكل نظيف وسريع ومناسب للجوال.

حساب GitHub تواصل معي
عدّل النصوص والروابط والإيميل قبل النشر النهائي.

نبذة

مسؤول تدريب مهتم بالبيانات، الأتمتة، تطوير الأنظمة، وتحسين تجربة العمل باستخدام أدوات رقمية حديثة.

الخدمات أو الاهتمامات

  • إدارة التدريب والتوثيق
  • تحليل البيانات ولوحات المتابعة
  • Power Query و Power BI
  • أتمتة الإجراءات وتحسين سير العمل

ملاحظات النشر
